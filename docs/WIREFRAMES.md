## Design Principles

### Color Palette
- **Primary**: `#667eea` (Purple)
- **Secondary**: `#764ba2` (Deep Purple)
- **Background**: `#f5f7fa` (Light Gray)
- **Text**: `#333333` (Dark Gray)
- **White**: `#ffffff`

### Typography
- **Headings**: Bold, 1.5rem - 2.5rem
- **Body**: Regular, 1rem
- **Labels**: Semi-bold, 0.9rem

### Components
- **Cards**: White background, rounded corners (15-20px), shadow
- **Buttons**: Gradient background, rounded (10-30px)
- **Inputs**: Border 2px, rounded (10px), focus state with color change

### Responsive Design
- **Desktop**: 1200px max-width
- **Tablet**: Stack cards vertically
- **Mobile**: Single column layout, adjusted padding
